Polyglot

Tools used:
  • Dart/Flutter
  • Python
  • Firebase
Tossing up between React Native and Flutter for creating a cross-platform app, I chose Flutter as the more efficient tool due to the gentler learning curve, extensive documentation and tutorials.
Summary
My largest and longest-running project, Polyglot is a cross-platform language learning app.
Motivation
I've always had a fascination with languages (hence studying a Bachelor's in linguistics) and was disappointed with the range of languages available on popular language-learning platforms such as Duolingo, Memrise, Babbel and Busuu. Polyglot is my response.
Technology
Written in Dart with Flutter and using Firebase for the backend, Polyglot integrates with third-party APIs (Google Image Search, Forvo) to provide images and audio to accompany target words. It uses a serverless architecture provided by Firebase. Language data is sourced from Wiktionary and language-specific dictionary PDFs using Python scrapers. The majority of the language data used in Polyglot is sourced from Wiktionary. I chose Wiktionary over other online sources for its open-source license and large dataset, especially for minority languages. Unfortunately, no suitable API is provided so I created a Python scraper. Scraping is not ideal as minor changes to the HTML of scraped pages can ruin the results and different languages often have different HTML layouts. However, since the scraper is only used during data collection, I am able to clean up the data before the user can access it in lessons.
